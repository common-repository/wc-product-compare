WC Product Compare is a product comparison plugin - user can compare from unlimited product comparison list and choose the right one.

== Description ==
WC Product Compare is a product comparison plugin - user can compare from unlimited product comparison list by adding them into compare list and choose the right one.

= Features =
WC Product Compare has following list of features:

* Enable/Disable the plugin from backend settings options.
* Enable product compare in shop page and single product page.
* Show Product Comparison list in Popup or page.
* Customize Compare button texts.
* Customize comparison product fields as you wants.
* Ordered comparison product fields as you wants.
* n lengths products can be added into comparison list.
* Remove products from comparison list.
* Empty product comparison list.
* Can use [wcpc_compare] shortcode to show product comparison list.
* Added so many hooks and filters for developers to customize as they wants.


I will definitely includes some more features in next. So If your required feature is not in this plugin, feel free to put a request in the support forum for this plugin, I will definitely take care of it. 

= Compatibility =
* WooCommerce 3.6+
* WordPress 5.2
* The theme should be WooCommerce standards compliant.
* Multilingual Support and is fully compatible with WPML.

== Installation ==
1. Upload the `WC Product Compare` folder to the `/wp-content/plugins/` directory OR search for "WC Product Compare" from your WordPress admin.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin settings through the "WC Product Compare settings" menu on your WordPress admin panel.
4. Now click on the settings link and configure the plugin as per your requirement.

This plugin is created for WooCommerce, so you must have to WooCommerce is installed in your WordPress website. 
If your WordPress website haven't WooCommerce installed then first install WooCommerce plugin please follow the link of WooCommerce plugin https://wordpress.org/plugins/woocommerce/. If you are new in WordPress and don't know to how to install a WordPress plugin then follow the link https://codex.wordpress.org/Managing_Plugins. 
You can find how to install WordPress plugin. For this plugin install you can also follow the same link.
WC Product Compare maintain the standardization of woo themes. There are we are working on the slandered hooks and filter for the given functionality.
